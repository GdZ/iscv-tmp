# iscv-tmp

## system require

1. axel
2. python
3. make

## some command

1. download dataset

```shell
make download
```

2. configuration of python environment 

```shell
make env
```

3. combination between depth and rgb

```shell
make associate
```

4. build project

```shell
make build
```
